DateTime widget


Focus on input and rotate the mousewheel or press arrow keys to scroll individual fields.

To use the DateTime widget:

  1. Include jQuery 1.3.2+
  2. Include jQuery UI 1.7+ with Datepicker
  3. Include the datetimewidget.js script
  4. Include the datetimewidget.css stylesheet
  5. Give the input elements the vDateInput / vDateTimeInput class
